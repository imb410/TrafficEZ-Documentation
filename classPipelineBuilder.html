int  hStackLength = 3

Preprocess image with the builder pattern. This is a slower implementation as compared to the process method. This is because we are creating a new frame for each process, then stacking it horizontally and vertically to show each step of the preprocessing defined by the builder.

Parameters
frameThe image frame to be processed.
hStackLengthThe max horizontal stack length for a tiled view of each step.

void PipelineBuilder::updateStepParameterById ( size_t  stepIndex,
int  paramId,
const std::any &  value 
)

This method provides a way to adjust just one parameter of a step without needing to update all parameters of the step. It's useful for fine-tuning individual aspects of a step's behavior during runtime.

Parameters
stepIndexThe index of the step in the pipeline to be updated.
paramIdAn identifier for the specific parameter within the step to update. The meaning of this identifier is specific to the type of step being modified.
valueThe new value to be assigned to the parameter identified by paramId. The type and value range of this parameter must be appropriate for the parameter being updated.

Detailed Description

Class for tracking convex hulls. Optionally, you can specify in the constructor the following:

Parameters
maxDiffDistanceMaximum distance difference for Centroid Tracking.
hullAreaThresholdArea threshold for hull similarity comparison.
boundaryCushionPixelsboundary line cushion (pixels) to start considering exit hulls.
maxFramesNotSeenMaximum number of frames a hull can be not seen before removal.
maxIdMaximum ID value for trackables before resetting the count.
